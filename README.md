# mini_project_squelize
  
  <h1> run on your database</h1>
  step 1: npm i</br>
  step 2: npx sequelize-cli db:migrate (migrate models)</br>
  step 3: npx sequelize-cli db:seed:all (demo data)</br>
  step 4: npm start (run pj)</br>
